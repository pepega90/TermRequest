# TermRequest

TermRequest is a terminal-based HTTP client built with Go.
It brings a Postman-like experience directly into your terminal using a clean and interactive TUI.

## ✨ Features

- 🔹 Send HTTP requests (GET, POST, PUT, DELETE, etc.)
- 🔹 Custom Query Params
- 🔹 Custom Headers (Authorization, Content-Type, etc.)
- 🔹 JSON Request Body support
- 🔹 Clean response viewer
- 🔹 Interactive Terminal UI powered by tview
- 🔹 Lightweight and fast
